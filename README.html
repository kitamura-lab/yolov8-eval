YOLO性能比較

機能

  • PCやOSごとにYOLOv8の性能比較を行う．

インストール(Windows11)

  • gitのインストール
    • ここからダウンロードし，インストールする．
  • リポジトリをクローンする．
git clone https://github.com/kitamura-lab/yolov8-eval.git
  • NVIDIAドライバのインストール
    • ここからGPUに対応する最新のNVIDIAドライバをダウンロードし，インストールする．
  • CUDAのインストール
    • ここからCUDA12.6をダウンロードし，インストールする．
  • Python3.12のインストール
  • pytorch-gpu.batの実行
    • 最後にTrueが表示されれば環境設定が終了．

インストール(Ubuntu)

  • gitのインストール
sudo apt install git
  • リポジトリをクローンする．
git clone https://github.com/kitamura-lab/yolov8-eval.git

  • CUDAのインストール

    • ここからCUDA12.6を手順に従い，インストールする．

  • 古いドライバーを削除する．

sudo apt --purge remove nvidia-*

  • NVIDIAドライバのインストール

    • Nouveauの無効化
    sudo gedit /etc/modprobe.d/blacklist-nouveau.conf

    で，以下を保存する．

    blacklist nouveau
options nouveau modeset=0

    以下を実行する．

    sudo update-initramfs -u
    • 推奨ドライバを確認する．recommendedのものを選択する．
    ubuntu-drivers devices
    • aptからインストール．XXXは推奨ドライバに置き換える．
    sudo add-apt-repository ppa:graphics-drivers/ppa
sudo apt update
sudo apt install nvidia-driver-XXX
    • 再起動
    • 確認
    nvidia-smi
    • 参考資料

  • Pythonのインストール

    sudo apt install python3-pip

  • 仮想環境作成

    python3 -m venv ../venv/yolo11
source ../venv/yolo11/bin/activate

  • パッケージのインストール

    pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu126
pip3 install ultralytics

  • 確認

    python3 check_gpu.py

実行

  • プログラムを実行する．
python3 eval_pose.py
